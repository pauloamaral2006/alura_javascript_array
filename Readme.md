## ℹ️ Sobre

<p>Projeto utilizado no curso para aprendizado das técnicas e melhores práticas.</p>

## 📘Ementa

### JavaScript: Arrays

- Compreender o que são arrays e porque são importantes
- Alterar arrays com os métodos do JS
- Entender o que são laços de repetição e como usá-los
- Modificar arrays utilizando métodos do JS moderno (ES6 em diante)
- Conhecer o spread operator e como o JS entende referências de arrays
- Praticar com os conceitos básicos e adquira conhecimentos sólidos
